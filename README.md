# Personal Blog

## Getting Started

1. Create a virtual environment.
    ```bash
    python -m venv .venv
    source .venv/bin/activate
    ```

2. Install requirements.
    ```bash
    pip install -r requirements.txt
    ```

3. Run the deployment server.
    ```bash
    python manage.py runserver
    ```
